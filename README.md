# Planning Poker

https://baemin.dev/planning-poker

## Development

### Prerequisites

- Bun
- Redis

### Install dependencies

```bash
bun install
```

### Run development server

#### API server

```bash
bun run dev:api-server # port 4111
```

> Redis server should be running on `localhost:6379`

#### Next.js server

```bash
bun run dev # port 3000
```
